使用Pelican和Github搭建博客

Sat 23 July 2016
By aReXnT

准备工作

  1. Github
  2. Markdown编辑工具 (MarkdownPad,Cmd Markdown)
  3. Windows下make工具

环境准备

  1. Python
  2. Virtualenv
  3. Pelican
  4. Git

搭建

安装Markdown和Pelican

virtualenv env
env\Scripts\activate
pip install pelican
pip install markdown

创建文件夹用于保存博客

mkdir blog
cd blog
pelican-quickstart

pelican-quickstart执行后，根据提示输入配置，除了部分必填基本上可以空（即默认）。这些配置都可以在pelicanconf.py进行更改，不用担心填错。

用Markdown写博文

markdown格式的源文件的头部必须按以下格式书写,具体参考文档定义。

Title: 文章标题
Date: 创建日期
Modified: 修改日期
Category: 文章分类，标志本文处于该分类下
Tags: 文章标签，标志本文处于该标签下
Slug: 
Author: 作者

把写好的.md格式的文件放到blog\content，然后回到blog目录下，执行make html命令进行博客的生成。

make html (pelican e:/blog/content/ -o e:/blog/output -s e:/blog/pelicanconfg.py)

make html命令将把刚才写的博文生成html，存放到output目录下，在windows下没有make，可以执行第二行的命令代替。 接着执行make serve开启测试服务器

make serve (cd e:/blog/output/ && python -m pelican.server)

安装主题

Pelican Themes: DEMO

在Github上的Pelican主题开源库

使用git clone把主题库拉到本地

git clone https://github.com/getpelican/pelican-themes.git

选择喜欢的主题，然后把文件夹复制到blog目录下，修改pelicanconf.py配置文件，添加主题参数

>THEME = 'pelican-themes/gum'

添加评论系统

博客在于分享，分享就需要交流。 在Disqus上申请帐号，按照流程Disqus会分配给你站点的Shortname，记牢Shortname，如果忘了请进入admin/settings中查看。然后同理，在pelicanconf.py添加

DISQUS_SITENAME = 'Shortname'
