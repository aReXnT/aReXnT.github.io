  1. 使用Pelican和Github搭建博客

    Sat 23 July 2016
    By aReXnT

    准备工作

    1. Github
    2. Markdown编辑工具 (MarkdownPad,Cmd Markdown)
    3. Windows下make工具

    环境准备

    1. Python
    2. Virtualenv
    3. Pelican
    4. Git

    搭建

    安装Markdown和Pelican

    virtualenv env
env\Scripts\activate
pip install pelican
pip install markdown

    创建文件夹用于保存博客

    mkdir blog
cd blog
pelican-quickstart

    pelican-quickstart执行后，根据提示输入配置，除了部分必填基本上可以空（即默认）。这些配置都可以在pelicanconf.py进行更改，不用担心填错。
